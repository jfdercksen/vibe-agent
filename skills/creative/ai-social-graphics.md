# Skill: AI Social Graphics

## Purpose
Produce platform-optimized static visuals for social media — LinkedIn graphics, Instagram carousels, YouTube thumbnails, quote cards, and infographics.

## When This Activates
When creating social media assets, when atomizing content into visual formats, or when building the creative variant set from AI Creative Strategist briefs.

## Inputs Required
- Creative brief or content to visualize
- Brand guidelines (colors, fonts, logo)
- Platform and format specifications
- Key message / headline
- Visual style direction

## Graphic Types

### Quote Card
**Purpose:** Share one key insight as a shareable image.
**Elements:** Quote text + attribution + brand element
**Platforms:** LinkedIn, Instagram, Twitter/X

### Carousel (LinkedIn)
**Purpose:** Educational or storytelling content in swipeable slides.
**Elements:** Cover slide + 6-10 content slides + CTA slide
**Rules:** Text-heavy is OK. Each slide = one point. Consistent design.

### Carousel (Instagram)
**Purpose:** Visual-first educational or storytelling content.
**Elements:** Cover + 5-8 slides + CTA
**Rules:** More visual, less text than LinkedIn. Bold typography. Consistent template.

### YouTube Thumbnail
**Purpose:** Drive clicks from search/browse.
**Elements:** Face (if personal brand) + bold text (3-5 words) + emotion/contrast
**Rules:** Readable at small size. High contrast. Curiosity gap.

### Infographic
**Purpose:** Visualize data, processes, or frameworks.
**Elements:** Title + visual data/steps + brand footer
**Platforms:** Pinterest (vertical), blog embeds, LinkedIn

### Ad Creative (Static)
**Purpose:** Stop the scroll and drive clicks.
**Elements:** Hook visual + headline + CTA + brand
**Rules:** One message. Clear CTA. Tested in variants.

## Process

### Step 1: Define the Visual Brief
For each graphic:
```markdown
## Visual Brief: [Asset Name]

**Type:** [Quote card / Carousel / Thumbnail / etc.]
**Platform:** [Where it will be posted]
**Dimensions:** [Width x Height]
**Key Message:** [The one thing the viewer should take away]
**Headline Text:** [Exact text on the graphic]
**Visual Style:** [Minimalist / Bold / Photographic / Illustrated]
**Color Palette:** [Primary, secondary, accent from brand guide]
**Typography:** [Headline font + body font]
**Must Include:** [Logo, URL, CTA, specific imagery]
```

### Step 2: Design Direction
Specify:
- **Layout:** Where text goes, where visuals go, hierarchy
- **Imagery:** Photo-based, icon-based, abstract, or typography-only
- **Mood:** Professional, playful, urgent, calm, bold
- **Reference:** "Similar energy to [example]" if helpful

### Step 3: Generate with Glif MCP
If using Glif MCP for generation:
- Provide detailed prompt with all visual specs
- Generate 3-5 variants per concept
- Vary: color emphasis, layout, imagery style

### Step 4: Carousel Structure (if applicable)
```markdown
## Carousel: [Title]

Slide 1 (Cover): [Hook / Title — must make them swipe]
Slide 2: [First point — set the context]
Slide 3: [Key insight 1]
Slide 4: [Key insight 2]
Slide 5: [Key insight 3]
Slide 6: [Example or proof]
Slide 7: [Summary / takeaway]
Slide 8 (CTA): [What to do next — follow, link, comment]
```

### Step 5: Quality Review
- [ ] Text is readable at actual display size (zoom out and check)
- [ ] Brand colors and fonts used correctly
- [ ] One clear message per graphic
- [ ] No typos (triple-check all text)
- [ ] CTA is present where appropriate
- [ ] Consistent style across a set of related graphics

## Platform Dimension Reference
| Platform | Format | Dimensions | Notes |
|----------|--------|-----------|-------|
| LinkedIn post | Square | 1200x1200 | Text-friendly |
| LinkedIn carousel | PDF slides | 1080x1080 or 1080x1350 | Upload as PDF |
| Instagram feed | Square | 1080x1080 | Visual-first |
| Instagram carousel | Square | 1080x1080 | 2-10 slides |
| Instagram Story | Vertical | 1080x1920 | Full-bleed |
| Twitter/X | Landscape | 1200x675 | Single image |
| YouTube thumbnail | Landscape | 1280x720 | High contrast |
| Pinterest | Vertical | 1000x1500 | Long-format |
| Facebook ad | Landscape | 1200x628 | Minimal text (20% rule) |

## Quality Gates
- [ ] Dimensions match target platform
- [ ] Readable at actual display size (not just zoomed in)
- [ ] Brand guidelines followed (colors, fonts, logo placement)
- [ ] One message per graphic
- [ ] Carousel has a strong cover slide that earns the swipe
- [ ] Human approves before publishing
